# 운명비서 콘텐츠 헌법 1.0

랜딩 결과 카피 구현: `src/lib/landing-insight-copy.ts`

## 1. 운명비서의 정체성

운명비서는 단순 운세 서비스가 아니다.

운명비서는 사용자가 매일 자신의 선택, 감정, 관계, 돈, 행동 패턴을 들여다보게 만드는 자기이해형 운세 서비스다.

사용자는 오늘의 운세를 보러 들어오지만, 실제로는 "왜 나는 또 이렇게 했을까?"에 대한 답을 원한다.

## 2. 핵심 감정

위로가 아니라 소름. "맞네"가 아니라 "이걸 어떻게 알았지?"

## 3. 절대 금지

- 오늘은 좋은/조심하세요, 당신은 ~한 사람입니다, 긍정적으로/걱정하지 마세요, 좋은 기회 등 범용·위로형 문장

## 4~5. 좋은 문장 · 공식

- 성격 설명 금지 → 행동 뒤 숨은 이유 폭로
- 공식 A 착각 깨기 / B 숨은 이유 / C 모순 / D 위험한 순간

## 6. 결과 카드 원칙

**4단 구조를 유지한다.** 라벨과 역할은 아래와 같다.

| # | 라벨 | 역할 (질문) |
|---|------|-------------|
| 1 | 오늘의 흐름 | **운세** — 오늘 어떤 흐름이 오는가 |
| 2 | 행동하기 좋은 타이밍 | **타이밍** — 언제 움직일 것인가 |
| 3 | 실수할 가능성이 높은 순간 | **위험 장면** — 어디서 실수하는가 |
| 4 | 오늘 비서의 제안 | **행동** — 그래서 오늘 뭘 하면 좋은가 |

구현: `src/lib/landing-insight-copy.ts`

- **1~3**: 운세 톤. `오늘은 ~하기 쉽다` / `~에 흔들릴 수 있다` / `~하는 편이 유리하다` / `~하는 편이 좋다`
- **4**: 구체적 행동 제안만. 명언·교훈·감성 글 금지
- **성격 낙인 절대 금지**: 당신은 ~한 사람이다 / 게으르다 / 소심하다 / 결정을 못한다
- 성공 기준: "맞네" (X) → "이거 나잖아" (O)

## 16. 4단 구조 · 성격 낙인 회귀 금지

운명비서는 앞으로 아래 **4단 구조**를 바꾸지 않는다.

1. **운세** (오늘의 흐름) — 오늘 어떤 흐름이 오는가
2. **타이밍** (행동하기 좋은 타이밍) — 언제 움직일 것인가
3. **위험 장면** (실수할 가능성이 높은 순간) — 어디서 실수하는가
4. **비서의 제안** (오늘 비서의 제안) — 그래서 오늘 뭘 하면 좋은가

**절대 금지 — 회귀하지 않는다:** `당신은 ~한 사람입니다` / `당신은 ~한 사람이다` 형태의 성격 낙인. 카드 1~4 어디에도 쓰지 않는다.

## 7~10. 섹션별 작성법

§7 오늘의 운세: 패턴 + 오늘 드러남 + 방향 (운세 조언 금지)

§8 좋은 시간: 시간 + 패턴이 강해지는 이유 + 그 시간에 피할 것

§9 피해야 할 행동: 위험한 생각(공식 D) + 숨은 이유 + 오늘 위험

§10 한 문장: 명언·교훈 금지, 캡처용만

## 11. 말투

부드우나 날카롭게. 위로하되 달래지 않음. 점쟁이·상담사·자기계발·AI 균형 문장 금지. 짧게, 한 문장 한 찌르기.

## 12. 중독성

매일 건드릴 주제: 후회, 불안, 인정욕구, 통제욕구, 손실회피, 관계, 돈, 결정 습관, 미룸, 반복 실수

## 13. 유료 전환

무료 = 오늘의 나(소름 입구). 유료 = 왜 계속 이러는가(반복 패턴 해석).

## 14. 검수

1~2번(범용·좋은 말) 해당 시 폐기. 3~6번(찌름·숨은 이유·캡처·어떻게 알았지) 해당 시 사용.

## 15. 초능력 금지

미래 사건을 단정하지 않는다. 사건 예언 서비스가 아니라 반복 선택·감정·관계 패턴 해석 서비스다.

금지 예: 연락이 온다 / 싸움이 생긴다 / 누군가 당신을 생각한다 / ~하기 좋은 시간(행운 단정)

허용: 이 시간에는 답장을 미루면 불안해진다 / '괜찮아'가 먼저 나오기 쉽다 (내 패턴)
