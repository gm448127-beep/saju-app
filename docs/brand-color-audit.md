# 운명비서 색채 심층 진단 — 적용 요약

원문: `운명비서 앱 색채 사용 심층 진단.pdf` (스크린샷 픽셀 기반)

구현: `src/styles/brand-tokens.css` · `src/lib/brand-colors.ts`

## 결론 (원문과 동일)

- **리브랜딩 불필요** — 아이보리·골드 브라운·딥 브라운 골격 유지
- **최우선** — 마이크로카피·인터랙티브 보더 대비
- **역할 분리** — CTA 골드 vs 차트/점수 브라운

## 서피스 계층 (반영됨)

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--paper` | `#F5F0E8` | 페이지 베이스 |
| `--paper-subtle` | `#FAF6EF` | 인셋·서브 패널 |
| `--paper-elevated` | `#FFFDFC` | 카드·입력 필드 |
| `--paper-inset` | `#F7F2EC` | 구획 인셋 |
| `--paper-selected` | `#F2EBE3` | 선택 배경 |
| `--paper-selected-strong` | `#EADFCF` | 강한 선택 |

## 역할 분리

| 역할 | 토큰 | 용도 |
|------|------|------|
| 다크 primary | `--primary-dark` `#2F282B` | 제목·주요 다크 버튼 |
| 골드 CTA | `--accent` `#7A5D35` | 세컨더리 골드 버튼 |
| 데이터 | `--data-accent` `#C79A5A` | 그래프·점수 (CTA 아님) |
