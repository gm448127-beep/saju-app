# landing-restart — 재시작 타겟 랜딩

이혼 후 다시 시작하는 30~40대 여성용 출시 알림 페이지.

## 로컬 (메인 앱)

http://localhost:3000/landing-restart

## 로컬 (독립 앱)

```bash
cd landing-restart
npm install
npm run dev
```

http://localhost:3003

## Google Form

`.env.local`:

```
NEXT_PUBLIC_GOOGLE_FORM_EMBED_URL_RESTART=https://docs.google.com/forms/d/e/여기ID/viewform?embedded=true
```
